# 🧠 Multithreading Simulator

## 📌 Description
This project is an interactive Multithreading Simulator that demonstrates core Operating System concepts such as thread models, CPU scheduling, synchronization, and deadlocks.

It provides a visual and practical understanding of how threads behave and interact in multi-threaded environments.

---

## 🚀 Features

- 🔀 Thread Models Simulation
  - Many-to-One
  - One-to-One
  - Many-to-Many

- ⏱ CPU Scheduling Algorithms
  - First Come First Serve (FCFS)
  - Round Robin

- 🧵 Thread State Visualization
  - READY, RUNNING, BLOCKED, TERMINATED

- 🔐 Semaphore Implementation
  - Controls access to critical sections

- 🏢 Monitor (Producer–Consumer Problem)
  - Demonstrates safe synchronization using locks & conditions

- ⚠️ Deadlock Simulation
  - Shows resource waiting and blocking situations

- 📊 Performance Metrics
  - Throughput
  - Average Waiting Time
  - CPU Utilization

- 📈 Gantt Chart
  - Visual representation of thread execution

---

## 🛠 Tech Stack

- Language: Python  
- Libraries: Streamlit, Matplotlib, Threading  
- Tools: VS Code  

---

## ▶️ How to Run

1. Install dependencies:
pip install streamlit matplotlib

2. Run the simulator:
streamlit run rtmts.py
